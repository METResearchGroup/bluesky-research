# Superposter constructive analysis

We're interested in seeing if superposters posted more constructive content than non-superposters.

Some avenues to look at:

- Average constructiveness between superposters and non-superposters
  - Across all posts
  - Across all posts used in feeds

Step 1: Run the `create_all_superposters_view.py` file to create a view for all superposters.
